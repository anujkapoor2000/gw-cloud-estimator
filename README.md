Here's what's built across the 6-step wizard:
Step 1 — Project Setup: Client name, GW version (7.x → Landmark), contract type (T&M / Fixed / Hybrid), currency (GBP/USD/EUR/INR), margin % and contingency % sliders.
Step 2 — Module Scope: Card-based selector for PC, CC, BC, Jutro, DataHub, Integration — each shows typical week ranges and roles.
Step 3 — Requirements: Per-module inputs — complexity picker (Low/Med/High/Critical), sliders for custom screens, LOBs, integrations, Gosu LOC, test cases. Global migration type (Lift & Shift / Re-configure / Re-build), environments, parallel run weeks.
Step 4 — Rate Card: Fully editable table of 16 roles × internal + external daily rates. Pre-populated with realistic GBP values.
Step 5 — AI Accelerators: Toggle AI Testing (−30%), Tech Debt Radar (−20%), Requirements Analyser (−15%), Test DataHub (−10%). Shows combined saving and tool investment.
Step 6 — Dashboard with 6 sub-views:

Overview — module effort split bars, phase cost bars, T&M vs Fixed Price side-by-side, cost breakdown pie
SurePath Phases — Inception / Elaboration / Construction / Transition / Hypercare with days, cost and internal vs external comparison
Module Breakdown — per-module effort split (Dev, Test, BA, PM, Arch, DevOps) with complexity badges
Resource Model — full role × days × peak FTE × phase × internal/external cost table
Testing & QA — testing effort by module, AI testing impact, test strategy by phase, Tech Debt analysis
Commercial — price build-up ladder, margin analysis, contract recommendation, milestone payment schedule
